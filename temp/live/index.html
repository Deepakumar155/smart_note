About Smart Collab Editor

Smart Collab Editor is a real-time collaborative code editor with built-in notes. Multiple users can work together seamlessly on the same document, with instant updates. The editor is designed for developers, students, and teams who want to code, share, and collaborate effectively.

The project uses Node.js, Express, Socket.IO, MongoDB, and CodeMirror to deliver a smooth collaboration experience.
